Pipeline Architecture

Tentang Proyek

Sistem otomasi konten YouTube yang dijalankan dari awal hingga akhir tanpa intervensi manual. Scrape video sumber dari channel target, clip highlight sesuai niche (outdoor/survival & satisfying), generate subtitle karaoke dengan word-level timestamps (whisper), dan upload terjadwal ke YouTube.

Multi-channel: saat ini melayani PZenatama (outdoor/survival) dan BrainCandy (satisfying/animals). Setiap channel punya pipeline terpisah dan jalan 1x/hari via cron, 23:00 WIB.

Pipeline Flow

  • 1. Scrape video dari channel YouTube target (yt-dlp)
  • 2. Filter video berdasarkan durasi & relevansi niche
  • 3. Clip highlight segment dengan FFmpeg
  • 4. Generate subtitle karaoke (word-level whisper timestamps)
  • 5. Overlay subtitle ke video (FFmpeg)
  • 6. Upload terjadwal ke YouTube (YouTube Data API v3)
  • 7. Status & error notifikasi via Telegram

Fitur Utama

  • → Multi-channel (setiap channel punya config terpisah)
  • → Karaoke subtitle dengan word-level timing
  • → Auto-upload via YouTube Data API v3
  • → Cron-based scheduling (1x/hari, WIB)
  • → Error alert via Telegram bot
  • → Full remote control (Telegram DM)

Tech Stack

Python Whisper yt-dlp FFmpeg YouTube Data API v3 Telegram Bot
